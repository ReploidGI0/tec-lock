El código PIN, un método que consiste en un código numérico de 4 a 17 dígitos que al ser introducido nos permitirá desbloquear el smartphone. Al configurar un PIN, también puedes configurar el dispositivo para que te pida que introduzcas el PIN después de que el dispositivo se bloquee. Esto evitará que el dispositivo se desbloquee después de estar inactivo sin introducir primero el PIN.

En qué consiste

Este método consiste en realizar una cadena de al menos 4 o 6 dígitos de nuestra elección en el orden que el usuario quiera, sin embargo esta clave no se nos debe de olvidar ya que se nos solicitará cada vez que se apague la pantalla del dispositivo.

Como se configura

Estos ajustes se tomaron de un Samsung A32 Ajustes > Pantalla de bloqueo > Tipo de bloqueo de pantalla > Seleccionamos “PIN” > Ingresamos el PIN > Nuevamente se ingresa el PIN para confirmar > Damos clic en guardar.

Ventajas
  • Es práctico de utilizar
  • Una buena secuencia de números es difícil de descifrar
Desventajas
  • Se puede descifrar la contraseña fácilmente si es un código muy sencillo como 1234, fechas de cumpleaños, etc
